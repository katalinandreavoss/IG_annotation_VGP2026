# Genomic analysis of immunoglobulin loci in the Vertebrate Genome Project

The repository contains data and scripts for analysis of immunoglobulin (IG) loci in genomes of 531 species assembled by the Vertebrate Genome Project.

## Data
``all_species_stats_pruned_12052025.csv`` Annotation overview of all 531 species
``annotation_stats.zip`` Annotation statistics per class
``gene_fasta_files_032726.zip`` fasta files per class with gene annotations
